1. Introduzione

G03MySimpleIRTool è un'applicazione Java che, dato un corpus di documenti testuali presenti in una directory, consente di elencare quelli più rilevanti rispetto ad una query costituita da una o più parole.

Il criterio di rilevanza da adottare attribuisce ad un documento uno score tanto più alto quanto maggiore è la frequenza di occorrenza di ciascuna parola che costituisce la query all’interno del documento.

L’applicazione G03MySimpleIRTool è un esempio semplificato di sistema di Information Retrieval (IR).

2. Funzionalità di base

2.1. Selezione della directory

Fare click sul pulsante Apri cartella... per selezionare la directory contenente i documenti da analizzare. L'applicazione supporta solo file di testo a formattazione semplice (*.txt).

Top

2.2. Sottomissione della query

Inserire una query costituita da una o più parole (token) e fare click sul pulsante Esegui query per avviare l'elaborazione della query e la generazione dei risultati. L'elenco dei documenti sarà presentato in ordine di rilevanza decrescente rispetto alla query formulata. Per nome del documento s'intende il nome del file.

Top

3. Funzionalità avanzate

3.1. Visualizzazione dei documenti

Fare click sul pulsante Visualizza nel pannello dei risultati per visualizzare i documenti trovati utilizzando lo strumento di visualizzazione integrato. Facendo doppio click su un documento nel pannello "Vista directory" è possibile visualizzare in qualsiasi momento i documenti della collezione.

Top

3.2. Visualizzazione delle statistiche

Fare click sul pulsante Statistiche nel pannello dei risultati per visualizzare le statistiche sul documento. Le statistiche sulla collezione di documenti sono sempre visibili nell'apposito pannello dell'interfaccia grafica.

Top

3.3. Preferenze dell'utente

Attraverso la finestra di dialogo "Preferenze", disponibile nel menu "Modifica" è possibile accedere alle impostazioni dell'applicazione. La finestra di dialogo "Preferenze" è suddivisa in tre tab e consente all'utente di attivare e configurare diverse funzionalità aggiuntive quali il contributo allo score differenziato per il titolo e il corpo del documento, il filtraggio delle stopwords con selezione di un file esterno contenente la lista delle parole da filtrare e la possibilità di scelta di un modello IR alternativo (tf-idf).
